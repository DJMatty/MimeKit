DefaultRootCertificatesPath Field

The default root certificate store path.

Syntax

protected static readonly string DefaultRootCertificatesPath

Remarks

On Microsoft Windows-based systems, this path will be something like C:\Users\UserName\AppData\Roaming\mimekit\root-certs.crt.

On Unix systems such as Linux and Mac OS X, this path will be ~/.mimekit/root-certs.crt.

DefaultUserCertificatesPath Field

The default user certificate store path.

Syntax

protected static readonly string DefaultUserCertificatesPath

Remarks

On Microsoft Windows-based systems, this path will be something like C:\Users\UserName\AppData\Roaming\mimekit\user-certs.p12.

On Unix systems such as Linux and Mac OS X, this path will be ~/.mimekit/user-certs.p12.
