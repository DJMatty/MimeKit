AttachmentCollection Class

A collection of attachments.

Syntax

public class AttachmentCollection : ICollection<MimePart>, IEnumerable<MimePart>

Remarks

The MimeKit.AttachmentCollection is only used when building a message body with a MimeKit.BodyBuilder.
