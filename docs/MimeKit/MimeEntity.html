Register Method

Registers the custom MIME entity. Once registered, all MimeKit.MimeParser instances will instantiate your custom MimeKit.MimeEntity when the specified mime-type is encountered.

Syntax

public static void Register (string mimeType, Type type)

Parameters

mimeType
The MIME type.
type
A custom subclass of MimeKit.MimeEntity.

Exceptions

Type Reason
ArgumentNullException

mimeType is null.

-or-

type is null.
ArgumentException

type is not a subclass of MimeKit.Multipart, MimeKit.MimePart, or MimeKit.MessagePart.

-or-

type does not have a constructor that takes only a MimeKit.MimeEntityConstructorInfo argument.

Remarks

Your custom MimeKit.MimeEntity class should not subclass MimeKit.MimeEntity directly, but rather it should subclass MimeKit.Multipart, MimeKit.MimePart, MimeKit.MessagePart, or one of their derivatives.

Requirements

Namespace: MimeKit
Assembly: MimeKit (in MimeKit.dll)
Assembly Versions: 0.18.0.0
